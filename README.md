**Job 01: Create an Android project develop the features based on the following
user stories.**
Job Specification Information:
1. Create a project named Name_ID_REST01 in Android Studio.
2. Develop a ProductActivity screen as per the wireframe.
3. Develop relevant XML UI design for the recyclerview rows.
4. Add a REST Client library as per your choice as the dependency in your
build.gradle file
5. Add an Image Loading and caching library as per your choice as the
dependency in your build.gradle file

6. Consume the given API via the REST client of your choice from previous
step.
7. Parse the JSON Data
8. Display the parsed JSON Data in the UI as per the wireframe.
9. Add necessary permission in application manifest
10. Handle internet connectivity as per the requirement in the user stories.
11. Develop the button click event and implement Retry feature as per user
story.
